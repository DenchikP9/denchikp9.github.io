# 📖 Описание

**PlayerHUD** — мощный динамический HUD-плагин для SCP:SL.  
Показывает игроку:

- Ник + значки (создатель, владелец, админ эффекты)
- HP, AHP, Hume Shield
- Роль + цветной unitName
- Статус раунда
- Статус боеголовки
- Статус Dead Man Switch
- Генераторы SCP-079
- TPS сервера
- FF
- Количество игроков
- Лобби-информацию (таймер, блокировка)
- Часы, показывающие реальное время игрока

И всё это — в реальном времени через **HintServiceMeow**.

# ⚙️ Конфигурация

```yaml
playerhud:
  # Activating plugin
  is_enabled: true
  # Activating debug mode
  debug: false
  # Makes text gradient
  use_gradient: false
  # Allows you changing main HUD gradient text color by using HEX format
  gradient_first_color: ''
  gradient_second_color: ''
  # Allows you add role groups to HUD with their custom color
  role_styles:
    example1:
      color: '#C50000'
      donate_role: false
    example2:
      color: '#DC143C'
      donate_role: false
    donaterole:
      color: '#FFD700'
      donate_role: true
    example3:
      color: 'default'
      donate_role: false
  # Allows hide or show some information in HUD
  show_admin_effects: true
  show_mute_status: true
  show_remote_access: true
  show_creator_badge: true
  show_owner_badge: true
  show_warhead_status: true
  show_gen_status: true
  show_friendly_fire: true
  show_players_count: true
  show_hp: true
  show_tps: true
  show_clock: true
  # Allows to change HUD color text by using HEX format
  hud_color: '#e91e63'
  # The server name that will be shown at the bottom of the HUD
  server_name: 'changeme'
```

# ❓ FAQ

### HUD не показывается  
— Убедитесь, что HintServiceMeow установлен  
— `is_enabled = true`  
— Нет ошибок в логах  

### Цвета сломались  
Отключи градиент:
```
use_gradient: false
```

### Хочу изменить цвет HUD  
```
hud_color: "#ff00ff"
```
